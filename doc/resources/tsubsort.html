TSUBSORT

Syntax

Envariable

M2H_TSUBSORT=[ 0 | 1 ]

Element(s)

<TSUBSORT>
<NOTSUBSORT>

Command-line Option(s)

-tsubsort
-notsubsort

Description

TSUBSORT, when on, instructs MHonArc to sort threads by subject.

Default Setting

Off

Resource Variables

N/A

Examples

None.

2.1.0
